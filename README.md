# CPU 壓痕智慧檢測平台（CPU Imprint Inspector）

以富士感壓紙壓痕影像評估 CPU 散熱模組接觸品質的**規則式白盒檢測工具**。
單一 HTML 檔、瀏覽器離線執行、**所有運算與影像皆在本機，不上傳任何資料**。

- **線上使用**：開啟 GitHub Pages 網址即可（亦可下載 `index.html` 離線使用）
- **操作手冊**：[manual.html](./manual.html)

## 方法聲明（與論文一致）
- 判定核心為**規則式白盒分類器**：紅度/亮度雙門檻量化接觸顯色＋上/下/左/右/中五區覆蓋率，任一區空白率 > 20% 判為缺陷；**推論階段不含任何神經網路**。
- 「AI」指以 AI 代理輔助之**門檻校準**（座標下降＋隨機擾動，最小化 1−F1，收斂即停止）；介面之「收斂曲線」為此目標函數之下降過程，**非深度學習訓練損失**。
- **溫度梯度圖／熱流向量為物理模型推估**（T = Tmin + (1−ρ)(Tmax−Tmin)、q = −k∇T 之相對視覺化），**非實測溫度**。
- 紀錄完整性採 **SHA-256 雜湊鏈（單機 append-only＋輕量 PoW）**：可偵測報告產出後之事後竄改；屬單機機制、非分散式區塊鏈。
- 評估指標：Accuracy / Precision / Recall / F1 / IoU（IoU＝缺陷類 Jaccard＝TP/(TP+FP+FN)，集合層級）。

## 隱私
影像處理、資料庫（SQLite in browser）、雜湊鏈、加密匯出全部在瀏覽器本機完成；本站為純靜態頁面，無後端、無追蹤。
